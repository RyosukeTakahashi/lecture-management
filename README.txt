GoogleAppScriptで実装する、授業管理システム。

使用方法：

1.
https://docs.google.com/spreadsheets/d/19Z9nYpSRL-hECQPLMS50i_THJ2kUMYgo1IBojMpY-mI/edit#gid=1004063562
にアクセスする。
2.
File→Make a copy
で自分のGoogleDriveにファイルをコピーする。
3.
コピーしたシートで、
Tools→Script Editor
を開き、エディタを開くことができる。
必要に応じて編集
4.
必要に応じて授業記録用シートに記載されている科目名、生徒名を編集する。
5.
Form→View Live Formでフォームを確認できる。
このフォームのURLを配布し、記入させる。
GoogleDriveでフォームを共有すれば、スマホからでも、ホーム画面にアイコンを作りフォーム送信可能。
